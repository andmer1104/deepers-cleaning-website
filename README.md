# Deepers Cleaning Website

This project is a full stack website built for a cleaning business, focusing on handling user requests and managing data through a backend system.

The frontend allows users to submit service requests through a contact form, while the backend processes these requests and stores the data in a MySQL database.

## Features
- Frontend interface for service requests
- PHP backend for handling form submissions
- MySQL database for storing user data
- Input validation and error handling

## Technologies
- HTML, CSS, JavaScript
- PHP
- MySQL

## Status
The frontend is complete and functional. Backend improvements are in progress, including validation and security enhancements.
